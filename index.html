Урок 2.8 УСЛОВИЯ или условные инструкции в JavaScript

В JS есть условия или если более развернуто, то услвоные инструкции.
Давайте разберемся, что такое услвоие и условные инструкции, как они создаются, для чего и как ими пользоваться.

На самом деле между условием и условными инструкциями разницы нет, это одно и тоже.

Условные инструкции – это инструкции языка JavaScript (ECMAScript), которые выполняют определенные действия в зависимости от значения некоторого условия.

Инструкция (на английском statement) в отличие от выражения – это всегда то, что выполняет определённые действия.

Выражение – это всегда то, что возвращает значение.

Проще говоря, условие - это когда в зависимости от того верно ли условие, будет выполнен тот или иной блок кода.

Виды условных инструкций в JavaScript:

  • if (с одной ветвью);
  • if...else (с двумя ветвями);
  • else if... (с несколькими ветвями);
  • инструкция выбора switch.

Кроме условных инструкций в JavaScript имеется ещё тернарный оператор (?:).

Инструкция if

Инструкция if начинается с ключевого слова if, затем после пробела идут круглые скобки, в которых находится условие и далее блок инструкций в фигурных скобках.
При этом блок кода выполняется только в том случае, если условие, заданное в круглых скобках () истинно.

Проще говоря, инструкция if(...) вычисляет условие в скобках и, если результат true, то выполняет блок кода.

Синтаксис следующий:

// condition – это условие
if (condition) {
// блок кода, который выполняется один раз, если condition истинно
}

В качестве условия можно указывать любое выражение.
Если выражение приводится к истине (то есть, если Boolean(condition) === true), то блок код выполняется.
В противном случае нет.

Например:
let year = prompt('В каком году была опубликована спецификация ECMAScript-2015?', '');
if (year == 2015) alert( 'Вы правы!' );

В примере выше, условие – это простая проверка на равенство (year == 2015), но оно может быть и гораздо более сложным.
